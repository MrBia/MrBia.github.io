HTML

Html is a file to load all necessary javascript in the game

  • Define the game title
  • Loading phaser3 framework
  • Loading in-game objects
  • Loading fonts

JAVASCRIPT

This game imports two Javascript files.

  1. phaser.min.js
  2. My custom scripts

1. phaser.min.js is a Phaser3 library file that support to create game.

2. Custom script is the files that I created for game

  • preloader.js: file is called when before start game, you can load the assets for game here
  • game.js: file config for game
  • data.js: file contain data for each level
  • local-storage.js: file support for storage data 
  • sound-handle.js: file support for sound in game
  • all other javascript files is for  in-game objects

ASSETS

The assets is a folder where containing the images, fonts, and sounds for the game, you can easily customize this game by change the assets here

Introduction

Lucky Wheel is an interesting game. The game was made using the program Phaserjs 3. If you want to edit the game, you will need to have a license of the program. But to edit the images, just replace them.

INSTALLATION

To run this game:
1. Download a web server (I'm using XAMPP)
2. Go to xampp/htdocs create a new folder (eg: lucky-wheel)
3. Copy and paste the game folder there (assets, js, phaser, index.html)
4. Open the web brower
5. Open the game with link: localhost/lucky-wheel/index.html

HOW TO PLAY

Touch the tube to get the ball and touch the other tube to move the ball there. Only one ball can be moved on top of another if both balls are of the same color. You win if all the balls are sorted into the tube.
